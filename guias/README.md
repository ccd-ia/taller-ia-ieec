# Guías de referencia — Taller de IA con Claude Code (IEEC)

Estas son guías de **consulta** para usar después del taller. No son material de
clase: son referencias para tenerlas a la mano cuando trabajes sola o solo.

| Guía | Para qué sirve |
|------|----------------|
| [`claude-code-guia.md`](./claude-code-guia.md) | **La principal.** Referencia completa de Claude Code: el ciclo de interacción, administración del contexto, Plan Mode, memoria (CLAUDE.md), la carpeta `.claude/`, Skills, Commands, Hooks, Agentes, MCP, Plugins, selección de modelo, prompting con XML, privacidad de datos y handoff entre sesiones. |
| [`cheatsheet.md`](./cheatsheet.md) | Hoja de referencia rápida (1 página): comandos `/`, atajos de teclado, modos de permisos, administración de contexto y el flujo típico de trabajo. |
| [`cowork-guia.md`](./cowork-guia.md) | Guía de **Cowork**, la app de escritorio de Claude (sin terminal): automatizar archivos, generar documentos, tareas programadas. Útil para quien prefiere no usar la línea de comandos. |
| [`claude-chrome-guia.md`](./claude-chrome-guia.md) | Guía de **Claude in Chrome**, la extensión que deja que Claude vea y actúe en tu navegador (con tu aprobación): investigación web, scraping ligero, llenar formularios. |

> **Dignidad de los datos.** Todas estas herramientas envían a Anthropic lo que
> les muestras. Para tareas con datos sensibles del IEEC: mantén el dato local,
> humano en el bucle, y nunca lo saques del perímetro. La regla del taller aplica
> en las cuatro guías.

## ¿Por dónde empiezo?

- ¿Usas la **terminal** (Claude Code)? → `claude-code-guia.md` + `cheatsheet.md`.
- ¿Prefieres **sin terminal**? → `cowork-guia.md`.
- ¿Necesitas que Claude trabaje en el **navegador**? → `claude-chrome-guia.md`.
